To make your .so-object executable you need simply do the following :

  • Define a path to your system ld-linux.so
  • Target an entry point of your library with -Wl,-e

This simple code (let's call it executable_so) do the job

#include <unistd.h>

const char interp[] __attribute__((section(".interp"))) = "path-to-your-ld-linux.so"

void entry()
{
    _exit(0);
}

Often ld-linux could be found at /lib

Next you should compile it with

gcc -Wall -fPIC -shared -o test.so executable_so.c -lc -Wl,-e,entry
